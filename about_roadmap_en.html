Version/Feature Time Frame Comments
SEB 2.1.4 for macOS

Q3 2019

 Improves security and stability, final version of Config Key feature to check validity of settings used for an exam. The Config Key can be generated automatically on an exam server.
SEB 2.4 for Windows  Q4 2019 Config Key feature to check validity of settings used for an exam. Every SEB version generates the same Config Key, as long as the same settings (config files) are used. The Config Key will now be available in the macOS, iPadOS/iOS and Windows versions of SEB.
SEB 2.2 for macOS/iPadOS/iOS Q4 2019 Major release of SEB with a unified code base for macOS and iPadOS/iOS. Supports the Additional Resources feature, better process monitoring (Mac), binary security module and integration with SEB Server.
SEB 3.0 Base for Windows Q1/Q2 2020 Completely refactored SEB version for Windows using a Chromium browser engine: Improved web compatibility, better integrated, faster. Improved stability and security architecture. Integration with SEB Server.
SEB 3.1 Extended for Windows 2020 Completely refactored SEB version for Windows with full functionality of SEB 2.x, for example Additional Resources. Contains binary security module.
SEB 3.0 for macOS/iPadOS/iOS Q1/Q2 2020 Partially refactored SEB version using new WebKit 2 (WKWebView) browser engine for improved web compatibility, better performance and stability.
New SEB JavaScript API 2020 The SEB JavaScript API allows better and easier integration with exam and learning management systems.
Improved client monitoring 2020 Further security enhancements for improved monitoring of exam client devices which are connected to SEB Server or to an e-assessment system with built-in integration for these features (for example the new Signature Keys).
Accessibility improvements 2020 Improvements for visually impaired users. Built-in screen reader and improved support for third party tools.
