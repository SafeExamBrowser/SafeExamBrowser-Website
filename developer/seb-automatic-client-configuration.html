Automatic Client Configuration

This feature is currently supported by the iOS version of Safe Exam Browser, but is supposed to be available soon also in the other SEB versions. It is available in the Initial Configuration Assistant, which is displayed by SEB for iOS when the SEB app is started directly (using its icon) the first time or hasn't been configured with persistent client settings (which are then used everytime the app is started directly with its icon).

An institution can place an SEB Config File at a specific path on their web server. SEB will search those predefined paths (there are several variations possible) when the main URL of that institution is typed in. You can try to enter safeexambrowser.org, SEB will then be reconfigured to display the demo exam portal page.

To support Automatic Client Configuration for your institution, you must be able to place an SEB configuration file (saved for configuring client) named SEBClientSettings.seb in one of the following paths of your institution's web server:

  • seb.institution.org/safeexambrowser/
  • safeexambrowser.institution.org/safeexambrowser/
  • institution.org/safeexambrowser/
  • institution.org/.well-known/

Automatic Client Configuration will search all these paths in this order to find a SEBClientSettings.seb file and then configure the SEB client to use these settings.

Automatic Client Configuration works if you enter the institution's main URL in the text field Enter a URL of an institution which supports SEB. Main URL refers to a host domain URL with optional subdomains. In the examples above, you could also replace "institution.org" with "subdomain.institution.org" or "subsubdomain.subdomain.institution.org".

Automatic Client Configuration also works with the feature "Search Network" as long as the device is connected to a Wi-Fi network or using VPN with a cellular connection (VPN for this feature is supported starting SEB 3.1.1 for iOS). So when students are connected with your institution's network, they can configure SEB by just taping the "Search Network" button if you support Automatic Client Configuration.
